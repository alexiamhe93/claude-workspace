## Reasoning Toolkit — Logic Frameworks

**Key document**: Argument Validation and Verification Guide.md
Location: `YOUR_NOTES_PATH/Notes_master/Research projects/Logic/Argument Validation and Verification Guide.md`

Apply implicitly to all analytical work. Apply explicitly when asked to assess a claim or argument.

**Framework selection:**

| Situation | Framework |
|-----------|-----------|
| Research claims or hypotheses | Classical predicate + inductive logic |
| Psychological constructs (vague, gradual) | Fuzzy logic, many-valued logic |
| Updating beliefs with new evidence | Non-monotonic logic |
| Knowledge vs. belief distinctions | Epistemic logic |
| Philosophical/metaphysical arguments | Modal + relevance logic |
| Normative or ethical questions | Deontic logic |
| Apparent contradictions | Paraconsistent logic |
| Time-ordered reasoning | Temporal logic |
| Any complex argument | Argument Validation Guide: identify premises → formalise → test validity → assess soundness |

**Core principles:** charity, make hidden premises explicit, validity vs soundness, fuzzy framing for gradual concepts.
